### Clone this repository
```
git clone https://github.com/workleast/sftpgo-docker-compose.git
```
### First time setup
```
cd sftpgo-docker-compose
chmod +x setup.sh
./setup.sh
```
### Run SFTPGo container
```
docker compose up -d
```
